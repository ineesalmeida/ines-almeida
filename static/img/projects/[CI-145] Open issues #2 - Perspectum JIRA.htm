What should be the naming convention for of the Castor Reports?

The file name should be ProjectName_Date_Time_RecordsOutput.csv

For example if my project was AK-US-001-0101, and I was exporting it at noon on the 16th of July 2020, then it would be:
AK-US-001-0101_16-Jul-2020_1200_RecordsOutput.csv

To address in

Currently the Record ID is set to be the patient name.

What should be the record ID when there is no Patient for a Portal Study?

if a study has a patient/timepoint:

  • Castor Record ID = <QAS patient Name>

  • Castor Report Name = <QAS timepoint Name>_<AnalysisType>

if a study does not have a patient/timepoint:

  • Castor Record ID = <QAS Study ID (name)>

  • Castor Report Name = <QAS Study ID (name)>_<AnalysisType>

In the rare cases that these are non-unique, Anna has accepted that these can be manually edited before uploading to Castor.
